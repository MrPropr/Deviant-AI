# Ethics

This project studies jailbreak robustness while minimizing dual-use risk.

Core commitments:

- Public files contain sanitized examples only.
- Raw harmful prompts are kept in ignored local directories.
- Raw model outputs from harmful prompts are kept in ignored local directories.
- Public tables and figures report aggregate metrics, not operational details.
- The project evaluates robustness differences, not prompt recipes.
- Closed-source API experiments should respect provider terms and rate limits.
- Human review should avoid copying unsafe response text into public notes.

The initial scope excludes tool-using agents. AgentHarm is treated as related work rather than an initial dataset.
